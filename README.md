# 22 Floor

Одностраничный сайт для продажи декоративных свечей ручной работы с отдельной страницей реферальной программы. Проект сверстан на чистых HTML/CSS/JS и не требует бэкенда — корзина оформляет заказ через Telegram, поэтому всё работает статически.

## Основные возможности
- **Каталог с карточками продуктов** — данные берутся из массива `PRODUCTS` в `src/script.js`, каждой позиции соответствует изображение из каталога `foto/`.
- **Корзина с локальным хранением** — выбранные товары хранятся в `localStorage`, поэтому содержимое переживает перезагрузку страницы и возврат на сайт (до очистки куки/хранилища).
- **Оформление через Telegram** — после выбора набора кнопка «Оформить через Telegram» собирает текст с позициями и открывает чат `@ksenia_timofeevaa`.
- **Мобильная навигация** — внизу отображается фиксированное меню и плавающая кнопка корзины (на внутренних страницах), адаптация покрывает телефоны и планшеты.
- **Отдельная реферальная страница** — описывает условия 30 % скидки, даёт код и форму его активации, повторяя анимации и стиль главной.
- **SEO-атрибуты** — подключены `robots.txt`, `sitemap.xml`, OpenGraph-иконки и кастомные фавиконки.

## Структура проекта
```
├─ index.html          — главная страница и точка входа
├─ referral.html       — лендинг реферальной программы
├─ robots.txt / sitemap.xml
├─ foto/               — изображения товаров
└─ src/
   ├─ style.css        — общие стили, адаптив, анимации
   ├─ script.js        — каталог, корзина, Telegram-логика
   └─ referral.js      — генерация реферального кода и обработчики формы
```

## Как запустить локально
1. Склонируйте проект или скопируйте файлы в любую папку.
2. Запустите любой статический сервер (пример на Python):
   ```bash
   cd /путь/к/проекту
   python3 -m http.server 8000
   ```
3. Откройте `http://localhost:8000/index.html` для главной или `/referral.html` для реферальной страницы.

> Можно также просто открыть `index.html` двойным кликом, но некоторые браузеры блокируют работу `localStorage`/`window.open` из файловой схемы, поэтому локальный сервер предпочтительнее.

## Настройка и развитие
- **Продукты** — редактируйте массив `PRODUCTS` в `src/script.js`: `id`, `name`, `desc`, `price`, `image`. Файлы изображений кладите в `foto/` и обновляйте путь.
- **Стили** — все цвета, анимации и брейкпоинты находятся в `src/style.css`. Переменная `--background` задаёт базовый светлый фон, `--red`/`--deep-red` — фирменные оттенки.
- **Реферальные тексты** — блоки страницы `referral.html` легко переписать непосредственно в разметке; `src/referral.js` отвечает только за код и копирование.
- **Корзина** — логика добавления/удаления и синхронизации с `localStorage` реализована в функциях `changeItem`, `persistCart` и `restoreCart` (`src/script.js`). При необходимости очистки корзины вручную выполните в консоли `localStorage.removeItem('twentyTwoFloorCart')`.
- **Изображения/иконки** — фавиконки лежат в `src/22Floor.png`, иконка акции `src/sale.png` используется плавающей кнопкой на внутренних страницах.

## Полезные советы
- Проверьте сайт на разных ширинах: есть отдельные стили для `max-width: 768px` (мобильные) и десктопной версии.
- После изменений CSS перезапуска не требуется, просто обновите страницу.
- Перед публикацией убедитесь, что пути к изображениям корректны и все файлы присутствуют (особенно из папки `foto/`).
